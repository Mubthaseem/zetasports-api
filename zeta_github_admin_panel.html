GitHub Config & Access

How to get your GitHub Token:

To push files directly to GitHub from this panel, you need a Personal Access Token:

  1. Go to your **GitHub Settings > Developer Settings > Personal Access Tokens > Tokens (classic)**.
  2. Click **Generate new token (classic)**.
  3. Select the **"repo"** scope checkbox (this permits writing to files).
  4. Click **Generate Token** and copy the code. Paste it here and click **Save Config**.
